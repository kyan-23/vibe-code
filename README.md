# vibe-code

A generative music visualizer built with HTML, CSS, and JavaScript.

## What it does
- Reacts to microphone input, uploaded audio files, or a built-in synth.
- Creates animated particle systems, waveform ribbons, and frequency bars.
- Uses mood-based palettes and motion blur for a polished live visual experience.

## Run locally
1. Open `index.html` in your browser.
2. Allow microphone access if you choose the Mic source.
3. Upload an audio file or start the synth.
4. Adjust sensitivity, density, and blur for the best vibe.

## Files
- `index.html` — app UI and controls
- `styles.css` — responsive dark theme styling
- `visualizer.js` — audio processing and generative visuals
